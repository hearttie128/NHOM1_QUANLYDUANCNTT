## 🎯 Mục tiêu dự án
- Phân tích doanh thu theo nhiều tiêu chí
- Hiển thị dashboard trực quan
- Dự báo bán hàng bằng AI/ML
- Xuất báo cáo PDF/Excel
- Cảnh báo doanh thu bất thường

## ⚙️ Công nghệ sử dụng

### Frontend
- ReactJS + TypeScript
- Tailwind CSS
- Chart.js / Recharts

### Backend
- Node.js / Express
- RESTful API
- JWT Authentication

### Database
- PostgreSQL
- Redis

### AI/ML
- Python (scikit-learn, pandas)

### DevOps
- Docker
- GitHub Actions
- AWS / GCP
